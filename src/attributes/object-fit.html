object-fit: fill
object-fit: contain
object-fit: cover
object-fit: none
object-fit: scale-down 
    object-fit指定元素的内容应该如何适应所在的内容框
    类比于background: cover;

    object-fit: fill;
        拉伸内容以致宽高填充满内容框
    object-fit: contain;
        保持宽高比，缩放内容以致宽或高某一边紧贴内容框，将产生空白区
    object-fit: cover;
        保持宽高比，裁剪内容以致宽高填充满内容框
    object-fit: none;
        内容保留原有的尺寸
    object-fit: scale-down;
        内容的尺寸与 none 或 contain 中的一个相同，取决于它们两个之间谁得到的对象尺寸会更小一些
